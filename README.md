### Модуль 1 - Основы Go + Git

#### Игра "Угадай число" на Go

Простая консольная игра, в которой игроку нужно угадать случайное число от 1 до 100 за ограниченное количество попыток.

- При запуске игра генерирует случайное число в зависимости от сложности.
- У игрока есть N попыток, чтобы угадать число.
- После каждой попытки программа сообщает:
    - Если число меньше загаданного — "Секретное число больше👆"
    - Если число больше загаданного — "Секретное число меньше👇"
    - Если число угадано — сообщение о победе.
- Если попытки закончились, игрок проигрывает, и показывается правильный ответ.

В игре также присутствует

1. **Проверка ввода**
    - Обработка ошибок при вводе (если пользователь ввёл текст вместо числа).
2. **Подсказки по расстоянию**
    - Добавить уровни подсказок:
        - Разница ≤5: 🔥 "Горячо"
        - Разница ≤15: 🙂 "Тепло"
        - Иначе: ❄️ "Холодно"
3. **Запоминание прошлых попыток**
    - Выводить список введённых ранее чисел.
4. **Выбор сложности**
    - Easy: 1–50, 15 попыток
    - Medium: 1–100, 10 попыток
    - Hard: 1–200, 5 попыток
5. **Сыграть ещё раз**
    - После окончания игры предложить сыграть повторно без перезапуска программы.
6. **Сохранение лучших результатов**
    - Ведение файла `results.json`, где сохраняется дата, исход (победа/проигрыш), количество попыток.
